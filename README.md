## Random Programmer Quote
A simple web app that grabs and displays a random programming quote, but also allows you to get more quotes from the same author, 
by clicking the button with his/her name.

## How to Use
0. Make sure you have node.js installed on your system. You can find your applicable installer at [https://nodejs.org/](https://nodejs.org/)
1. In the terminal, clone the repository using `git clone https://www.github.com/actionjdjackson/random-programmer-quote.git`
2. Navigate to the directory named random-programmer-quote using `cd random-programmer-quote`
3. Install the necessary dependencies using `npm install` or `npm i` for short
4. Use nodemon or node to run the server using `nodemon index.js` or `node index.js`
5. Navigate your web browser to the localhost server on port 3000 by typing in `localhost:3000/` in the address bar
